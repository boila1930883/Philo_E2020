Philosophe du Jour :
	> Nicolas Machiavel :
			* Intellectuel italien du XVe siècle. Issu d'une famille noble. Il s'est ramassée comme conseiller politique, stratège, ...
			Il fut chassé d'un royaume où il était conseiller politique de l'ancien roi et afin d'y revenir pour y conseiller le nouveau dirigeant, il écrit un livre "Le Prince" qui explique comment conserver le pouvoir déjà acquis.

			"Le Prince" : ### Tous les Coups sont permis ###

							Comment fait-on pour acquérir/conserver le pouvoir ->

								a) Avoir une armée forte et nombreuse :
											- Bon moral (bons équipements, victoires, ...)
											- Bon patriotisme
								b) Utilisation des mercenaires :
											Aggrandi l'armée sur le coup, mais peut nuir si plus d'argent.
								c) L'envie :
											Bien paraître, avoir l'air du nation paisible, riche, confortable pour donner l'envie aux gens de joindre la nation.
								d) La Peur :
											- Être imposant pour donner l'envie aux opposants de se soumettre sans combattre
											- Faire de la Propagande/Campagne de Peur
												Ex.
													* Either you're with us, or either you're with the terrorists
													* La Famille Kin de Corée du Nord
								e) Le Bon Souverain :
											Si on commence à perdre le soutien du peuple, s'en assurer avant d'ordonner.
											Ex.
											 Ce mois-ci, congé de taxes pour tout le monde !

	Il faut s'assurer que le peuple considère le dirigeant comme supérieur (bon ou mauvais)

	Définitions :
		Machiavélique :
				Être prêt à tous les moyens pour prendre/conserver ce que l'on veut.

		Mercenaires :
				Soldats sans patriotisme, qui se battent pour de l'argent.

PLAN : DEVELOPPEMENT
	
	Syllogisme :
			Avec plusieurs informations, on peut en déduire une nouvelle.
			Ex. 
				Positif ->
					A = B
					B = C
					donc A = C

				Négatif ->
					A != B
					B = C
					A != C

	Prémisses :	Une information permettant de faire le syllogisme

		Ex.	Dans le système de référence du système solaire.
				La Terre tourne autour du soleil
				Je suis assise sur la Terre
				Je suis donc en mouvement

	Dans le Développement :
		* Deux prémisses qui permettent de tirer une conclusion qui permet de répondre à la question.
